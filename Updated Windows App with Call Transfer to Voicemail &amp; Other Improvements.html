Updated Windows App with Call Transfer to Voicemail & Other Improvements

A new Windows App release is now available for download and includes the ability to transfer calls to an extension’s voicemail, as well as some very important usability and connectivity fixes.

What’s Improved

The latest Windows App version delivers the ability to transfer calls to an extension’s voicemail from the “Receptionist”, “Q-Manager” or “Manager” view. This extends the transfer functionality while managing queue calls in one of the app’s dedicated views.

The new app also delivers these significant fixes on usability and connectivity:

  • Abandoned calls now display the queue name in views e.g. the queue manager.
  • While in an active call, the contact’s relevant “Company Name” is now shown on screen after answering the call.
  • Email notifications are sent to attendees when an audio conference is cancelled.
  • The “Callee” name is now displayed when a call goes to a remote extension over a bridge.
